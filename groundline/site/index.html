Groundline GraphDB

A powerful decentralized graph database with IPFS persistence,
CRDT-based collaboration, and seamless knowledge graph integration.

Get Started GitHub

Key Features

CRDT-Based Collaboration

Built on Yjs for real-time collaborative graph editing, enabling seamless team collaboration.

IPFS Integration

Decentralized storage with Helia and optional Filecoin pinning for persistent data storage.

Knowledge Graph Integration

Native support for Wikidata, DBpedia, and OpenAlex with seamless data import.

JSON-LD Support

Full semantic web compatibility with JSON-LD context for structured data representation.

Rich Graph Operations

Comprehensive API for graph manipulation, querying, and advanced operations.

Provenance Tracking

Track changes and maintain detailed version history of your graph data.

Quick Start

npm install @shubhamrasal/groundline

import { createGraphDB, createGraphIPFSManager } from '@shubhamrasal/groundline';

// Initialize graph database
const graphDB = createGraphDB({
  enabledAdapters: ['wikidata', 'dbpedia', 'openalex']
});

await graphDB.initialize();

// Import data from external knowledge graphs
const results = await graphDB.importExternalKG('wikidata', {
  query: 'your-sparql-query'
});

Requirements

  • Node.js >= 18.0.0
  • NPM or compatible package manager
