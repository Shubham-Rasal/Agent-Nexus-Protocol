Installation & Setup

1. Download

Get the latest release from our GitHub repository

2. Install

Extract to your vault's .obsidian/plugins directory

3. Configure

Enable in Community Plugins and add your Filecoin credentials

How to Use

Ribbon Icon

  1. 1 Click the cloud upload icon in the left sidebar
  2. 2 Choose "Backup Vault to Filecoin" or "Backup Current File"
  3. 3 Monitor progress in the notification panel

Command Palette

  1. 1 Open Command Palette (Ctrl/Cmd + P)
  2. 2 Search for "Backup vault" or "Backup current file"
  3. 3 Select the desired backup option

Automatic Backups

Enable auto-backup in settings and set your desired interval. The plugin will automatically backup your vault without any manual intervention.
